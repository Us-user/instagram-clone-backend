## Решения
- **Таргет `net8.0` при рантайме .NET 10.** На машине только SDK/рантайм 10.0.8. Собираем под
  `net8.0` (ТЗ — источник истины), а в `WebApi.csproj` добавлен `<RollForward>Major</RollForward>`,
  чтобы приложение запускалось на установленном рантайме .NET 10. Проверено: `dotnet run` стартует,
  Swagger отдаёт `v1/swagger.json` (HTTP 200, title «Instagram Clone API»).
- **`FrameworkReference` вместо пакетов Identity.** Чтобы версии Identity во всех слоях совпадали
  (никаких package-конфликтов), `IdentityUser<string>` берём из общего фреймворка ASP.NET Core.
- **AutoMapper 14.0.0 + подавление NU1903.** Advisory GHSA-rvv3-g6hj-g44x (CVE-2026-32933) — DoS
  через переполнение стека на глубоко вложенных/циклических графах. Патч только в коммерческих
  15.1.1+/16.1.1+ (AutoMapper с v15 платный). ТЗ требует AutoMapper → берём последнюю бесплатную
  (MIT) 14.0.0. Маппинги плоские Entity↔DTO без циклов — вектор неприменим; `NU1903` осознанно
  подавлен на уровне этого `PackageReference` (`NoWarn="NU1903"`) с комментарием.
- **XML-doc для Swagger** включён уже сейчас (`GenerateDocumentationFile`, `NoWarn 1591`).
